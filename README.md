## 🚀 Instalação e uso

```bash
# Clone o repositório
git clone https://github.com/kaerinDev/Todo-Ignite-Rocketseat.git

# Acesse a pasta do projeto
cd Todo-Ignite-Rocketseat

# Instale as dependências
npm install

# Execute a aplicação
npm run dev

# O servidor inciará na porta:5173 - acesse http://localhost:5173/
```
